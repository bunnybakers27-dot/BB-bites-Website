## 🔥 Firebase Database

This website uses **Firebase Realtime Database** (free Spark plan) to sync data across all devices in real time.

- When admin updates a product → all customers see it instantly
- Works across all phones, tablets and computers
- Free forever for small bakery usage

**Firebase Project:** BB-bites Website
**Database Location:** Singapore (asia-southeast1)
